# MEU PROJETO

<p>Utilizei Nest, Typeorm, Postgres e Redis como principais tecnologias para o desenvolviemnto da aplicação</p>

## INSTALAÇÃO

```bash
$ npm install
```

## ENVS

<p>É necessário criar um arquivo .env na raiz do projeto e adicionar as variáveis de ambiente do postgres para o banco de dados conectar corretamente com a aplicação.</p>

```bash
$ DB_HOST DB_PORT DB_USERNAME DB_PASSWORD DB_DATABASE
```

## PARA RODAR O APP

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
